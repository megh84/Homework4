# 3D Point Cloud Feasibility App

## Overview
This app analyzes manufacturing part feasibility using machine learning pipelines.

## Features
- Upload dataset_summary.csv
- Run multiple ML pipelines
- Compare F1 scores
- Identify best model

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Upload to GitHub and deploy using Streamlit Community Cloud.
